# Ejercicios propuestos – UD5: BOM y DOM

Colección de ejercicios prácticos para la Unidad 5 del módulo DWEC (2º DAW).

Estos ejercicios sirven para practicar el acceso y manipulación del DOM mediante JavaScript.
El último (Lista de la compra) es el mini-proyecto evaluable.

Cada carpeta incluye:
- `enunciado.md`: instrucciones.
- `index.html`: plantilla base.
- Archivos auxiliares (`script.js`, `style.css` si procede).
