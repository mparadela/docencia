### Sistemas Informáticos (SSII – DAW)
- **Currículo oficial:**  
  - Real Decreto 659/2023 y Orden ECD/841/2024 (Aragón).  
- **Documentación técnica:**  
  - *Operating System Concepts* (Silberschatz, Galvin, Gagne, Wiley).  
  - *The Linux Command Line* (William Shotts, No Starch Press).  
  - Manuales de *Ubuntu Server* y documentación de *Microsoft Learn*.  
- **Recursos docentes:**  
  - Apuntes de *Red Hat Academy* y *Cisco IT Essentials*.  
  - Materiales de *Aularagón* y de la comunidad docente de FP Informática de Aragón.
